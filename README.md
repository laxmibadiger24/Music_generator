CodeAlpha Music Generator

Project Description

This project is a simple AI Music Generator developed using Python. It generates random musical notes and saves them as a MIDI music file.

Features

- AI-based music generation
- Random musical note creation
- MIDI music file output
- Simple and beginner-friendly project

Technologies Used

- Python
- music21
- pygame

How to Run

Install required libraries:

pip install music21 pygame

Run the project:

python music_generator.py

Output

The program generates a MIDI music file named:

generated_music.mid
